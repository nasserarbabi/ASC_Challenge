Composites on Clouds currently support Universal Mesh (.unv) files. You can create them with SALOME, an open source plaftorm for numerical simulation. You should define different sections and node-groups in SALOME. You can later use them to apply boundary conditions and section properties in Composites on Clouds.

Since the 1st principal coordinate is assumed to be in the X direction, the mesh should be created in X-Y plane, with height of the part in the Z direction.

More information about the steps to create the mesh, partitions, and group-sets can be found in SALOME Documentation.
